# Sports Data Projects

This repository contains beginner data analysis projects created while learning Python, Jupyter Notebook, and sports analytics.

The goal of these projects is to practice working with datasets, performing basic analysis, and communicating results.

---

## Projects

### 1. Country Population Analysis
A beginner Python notebook exploring lists, loops, dictionaries, and functions using rounded country population data.

Skills practiced:
- Python lists
- loops and conditionals
- dictionaries
- basic classification logic

---

### 2. NHL Shot Distance Analysis (2021 Season)
This project analyzes how shot distance affects the probability of scoring a goal using NHL shot event data.

Skills practiced:
- data cleaning
- feature engineering
- grouping and aggregation
- visualization
- interpreting analytical results

Key question(s):
How does distance from the net affect scoring probability?
How do empty-net shots affect scoring probability as a whole?
